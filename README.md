## To build:

```bash
mvn compile 
```

## To run:

```bash
mvn exec:java
```

## To test:

```bash
mvn test
```

## To create JAR:
```bash
mvn package 
```

## To create Javadoc:
```bash
mvn install
```

The created JAR file will be found in the Game directory. An executable JAR file is already present in our Game directory. 
The generated Javadocs will be found in the Game/JavaDoc directory.
